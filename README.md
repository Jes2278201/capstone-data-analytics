# How the August Complex Fire affected the California Housing Market
This repository holds the files about my dive into understanding how the August Complex fire from August to November 2020 affected the housing market<br />
The Data Visualization folder holds my pbix file used to create my visualization in power BI<br />
The Datasets folder holds the xlsx file I used to create the visualizations and includes the original data, the modified format for the data, and whether the county was or wasn't affected by the fire<br />
The Documentation folder holds the docx file that records my progress, hypotheses, and conclusion to this project<br />
